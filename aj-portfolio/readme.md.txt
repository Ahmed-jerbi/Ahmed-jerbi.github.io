# SSG based on Jekyll

1. Install Ruby + devkit
> https://rubyinstaller.org/downloads/

2. Run the Ruby MSYS2 installer (cmd)
Choose 3. MSYS2 and MINGW development toolchain

3. Install jekyll
From the portfolio folder, enter cmd
> bundle install

4. Excecute server
> bundle exec jekyll serve --livereload
Or use the bath file "execute local server.bat"

5. Open the site in a browser
> localhost:4000
